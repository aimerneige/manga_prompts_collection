#### 【第 2 话】『屋上の風と鼓動』 (共4格)

##### 第1格
* **【构图与景别】**：全景 / 轻微仰视 / 门框构图
* **【可见场景与光影】**：教学楼通往天台的铁门被推开，门外是午后明亮的天空，过曝效果使门外景色呈白色高光。铁门内侧锈迹斑斑，地面为水泥灰色。
* **【动态视觉与定格姿势】**：
  - [画面中央偏左] 南小鸟：[当前完整穿搭：音乃木坂学院夏季校服——白色短袖水手服上衣，浅蓝色衣领，领巾为浅绿色。下身深蓝色高腰百褶裙，白色短袜，棕色乐福鞋。左手提着之前的浅黄色帆布袋，右手握着门把手，回头看向后方。] + [身体半转，重心在右脚，裙摆因转身而微微扬起。] + [表情开朗，嘴微张像在邀请。]
  - [画面右侧后方] 园田海未：[当前完整穿搭：同款夏季白色短袖水手服，浅蓝色衣领，领巾为浅蓝色。下身深蓝色高腰百褶裙，白色短袜，棕色乐福鞋。双手交叠在身前，微微低头。] + [站在门内阴影处，脚步略有迟疑。] + [表情略显紧张，眉毛微蹙，嘴唇抿着。]

* **【漫符与特效】**：小鸟周围有阳光洒入的星芒「ギラリ」，海未头顶有汗滴「ジトッ」。

##### 第2格
* **【构图与景别】**：中景 / 平视 / 人物居中
* **【可见场景与光影】**：天台场景，周围有生锈的围栏与通风管道。天空碧蓝，阳光从左上方照射，在地面投下短影。远处可见城市轮廓线。
* **【动态视觉与定格姿势】**：
  - [画面左侧] 园田海未：[当前完整穿搭：白色短袖水手服、浅蓝领巾、深蓝百褶裙。] + [背靠围栏站立，双手向后抓住栏杆，身体绷直，视线微微向下不敢看小鸟。] + [表情紧张，脸颊泛红，额发被风吹动。]
  - [画面右侧] 南小鸟：[当前完整穿搭：白色短袖水手服、浅绿领巾、深蓝百褶裙。] + [面对海未，身体前倾，双手背在身后，脸凑近海未的侧脸。] + [表情天真好奇，眼睛盯着海未的侧颜。]
* **【对白与气泡】**：
  - [顺位1] 南小鸟 ([云朵状气泡])：“海未ちゃんって、本当に高い所、苦手？”
  - [顺位2] 园田海未 ([颤抖线条气泡])：“だ、大丈夫です……少し風が強いだけで……”
* **【漫符与特效】**：风吹过表示「ビュウ」，海未身边有发抖线「ブルブル」。

##### 第3格
* **【构图与景别】**：近景 / 平视 / 聚焦上半身
* **【可见场景与光影】**：天空过曝成为纯白背景，两人脸部被柔光笼罩，无强烈阴影，显得干净通透。
* **【动态视觉与定格姿势】**：
  - [画面左侧] 园田海未：[当前完整穿搭：可见白色水手服衣领、浅蓝领巾细节。] + [姿势不变，但头转向小鸟一侧，黄绿色眼睛因近距离对视而动摇，瞳孔微微放大。] + [表情：脸颊红晕加深，嘴微张却说不出话。]
  - [画面右侧] 南小鸟：[当前完整穿搭：白色水手服衣领、浅绿领巾。] + [左手抬起，轻轻抓住海未的右手腕（抓在栏杆上的手），右手自然垂在身侧。] + [表情柔和，眼神温柔又坚定，像是给海未安全感。]
* **【对白与气泡】**：
  - [顺位1] 南小鸟 ([椭圆形气泡])：“私がついてるから、怖くないよ。”
